# Mimicopi Whistle
第 17 回 Siv3D Game Jam (テーマ「音楽アプリ」) 参加作品

#### 作者  
Reputeless (https://twitter.com/Reputeless)

#### 使い方
口笛と同じ音程の鍵盤が赤くハイライトされます。  
鍵盤はマウスクリックで鳴らせます。
